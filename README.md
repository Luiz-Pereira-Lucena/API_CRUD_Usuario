# API_CRUD_Usuario
É uma Application Programming Interface de cadastro de Usuarios.


### Neste projeto utilizamos:
* C#
* ASP.NET WEB API
* API REST
* Injeção de independências
* Migrations
* POO
* Repository
* Swagger
* Banco de Dados PostgreSQL
